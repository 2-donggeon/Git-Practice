# Shell Commands   
**How to Use Shell Commands**  
  
*First Step*  
Run Git Bash  
*Second Step*  
pwd : shows the current path in a hierarchical directory  
*Third Step*  
cd : change directory  
*Fourth Step*  
ls : list files and directories  
*Five Step*  
i : show detailed information  
*Six Step*  
exit : exit the program
